# 🚀 QuickHire – Job Portal Web Application

QuickHire is a modern job portal web application built to connect **job seekers** and **recruiters** through a simple, fast, and clean interface. It focuses on usability, performance, and a visually appealing dark-themed design.

---

## 🌟 Features

- 🔐 User Authentication (Login & Signup)
- 👥 Role-based system:
  - Job Seekers
  - Recruiters
- 📄 Post and manage job listings
- 🔍 Browse available jobs easily
- 🧑‍💼 Create and update professional profiles
- 🌍 Location support using Google Places API
- 🎨 Fully responsive dark-themed UI
- ⚡ Smooth and lightweight performance

---

## 🛠️ Tech Stack

### Frontend:
- HTML  
- CSS  
- JavaScript  

### Backend:
- PHP  

### Database:
- MySQL  

### APIs & Tools:
- Google Places API
